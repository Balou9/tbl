# tab-arr

***

Table arrays for frequency observation

***

## Get it!

```js
npm install --save tab-arr
```

***

## Usage

```js
var table = require('./index')
var array = ['Sophie', 'Max', 'Sophie', 'Max', 'Mike', 'Max', 'Sophie']
console.log(table(array))
```

***

## API

### `table(arr)`
